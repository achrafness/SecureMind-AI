# SecureMind AI


## What is SecureMind AI?

SecureMind AI is a unified, AI-powered cybersecurity platform that protects organizations against both network intrusions and email-based phishing attacks through advanced machine learning and real-time threat intelligence.

It combines:

- AI-Based Intrusion Detection System (IDS)
- AI-Powered Phishing Detection

## Dataset

This project uses the CICIDS2017 dataset (Canadian Institute for Cybersecurity) as the primary source of network traffic data for training and evaluating intrusion detection models. Key references and mirrors:


- Official dataset page: https://www.unb.ca/cic/datasets/ids-2017.html
